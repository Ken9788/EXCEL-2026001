# 第0-2章：3秒钟看见“引用”的变化

“在开始之前，请大家先打开Excel，在A1输入10，B1输入20，C1输入=$A$1+B1，向下拖动三行。看看发生了什么？这就是我们本章要讲的核心概念——单元格引用。”

## 现在就动手

打开Excel。

在A1单元格输入数字 **10**。

在B1单元格输入数字 **20**。

在C1单元格输入以下公式：

\=$A$1+B1

然后按回车。

你看到C1显示什么？是不是 **30**？

很好。现在把鼠标移到C1单元格的右下角，当光标变成黑色十字时，**向下拖动三行**（拖到C2、C3、C4）。

现在看一下结果：

| **单元格** | **显示的值** |
| --- | --- |
| C1  | 30  |
| C2  | 10  |
| C3  | 10  |
| C4  | 10  |

等等，C1是30，C2怎么变成了10？

💡 **在单元格里试试就知道了。**

看看C2的公式变成了什么？点击C2，看看公式栏：

\=$A$1+B2

再看C3：

\=$A$1+B3

发现了吗？

- $A$1 **纹丝不动**——无论拖到哪一行，它永远引用A1
- B1 **变了**——变成了B2、B3、B4

这就是为什么C2返回10（因为B2是空的，Excel把它当作0，所以10+0=10）。

**你亲眼看见了“引用”的两种行为：**

一个固定不变（$A$1），一个一直在变（B1）。

一个不动，一个跟着跑。

**这一章，就是让你搞明白：什么情况下该让它固定不变，什么情况下该让它跟着变化。**

## 一、函数的基本结构：一个公式是怎么组成的？

在深入引用之前，我们先理解一个公式的基本结构。

\=SUM(A1:A10)

拆解来看：

| **组成部分** | **示例** | **含义** |
| --- | --- | --- |
| 等号  | \=  | 告诉Excel：这是一个公式，请计算它 |
| 函数名 | SUM | 告诉Excel：我要做什么操作（求和） |
| 左括号 | (   | 参数列表的开始 |
| 参数  | A1:A10 | 告诉Excel：对哪些数据做操作 |
| 右括号 | )   | 参数列表的结束 |

如果公式有多个参数，用逗号分隔：

\=VLOOKUP(A2, 产品映射表!$A$2:$C$100, 3, FALSE)

这个公式有4个参数：

| **参数位置** | **值** | **含义** |
| --- | --- | --- |
| 第1个 | A2  | 找什么 |
| 第2个 | 产品映射表!$A$2:$C$100 | 在哪里找 |
| 第3个 | 3   | 返回第几列 |
| 第4个 | FALSE | 精确匹配 |

你不需要背这些。你只需要理解：**公式=等号+函数名+括号+参数**。每个函数的具体参数，你可以查，也可以问AI。

## 二、单元格引用——那个固定不变的和跟着变化的

**三种引用类型**

| **引用类型** | **写法** | **复制时行为** | **记忆口诀** |
| --- | --- | --- | --- |
| 相对引用 | A1  | 会随位置变化 | **跟着跑** |
| 绝对引用 | $A$1 | 始终不变 | **钉死了** |
| 混合引用 | $A1 或 A$1 | 一半钉死，一半跟着跑 | **钉一半** |

你刚才在练习中见到的：

- $A$1 → **固定不变**（绝对引用）
- B1 → **跟着变化**（相对引用）

**什么时候用哪种？**

**相对引用（跟着变化）**：当你想对每一行分别计算时。

\=销售额/花费

每一行都用自己的销售额和花费计算ROAS——这就是相对引用。

**绝对引用（固定不变）**：当所有行共用同一个固定值时。

\=花费/$E$1

其中$E$1存放固定目标ACOS（如25%），所有活动都用同一个目标值——这就必须用绝对引用。

**混合引用（只固定行，或只固定行列）**：当你需要复制公式到多行多列，但只希望部分变化时。

**F4键——最省事的切换方法**

在编辑公式时，选中单元格引用（如A1），按F4键可以快速循环切换：

按1次：A1 → $A$1 钉死了

按2次：$A$1 → A$1 行钉死，列活

按3次：A$1 → $A1 列钉死，行活

按4次：$A1 → A1 回到起点

**实战技巧**：写公式时，把引用部分选中，按F4切换到你想要的方式。不用背，多试几次就形成肌肉记忆了。

## 三、文本与数值——一个容易踩的坑

在Excel中，文本和数字看起来可能一样，但Excel对待它们的方式完全不同。

💡 **在单元格里试试就知道了。**

在A1输入：'001（前面加英文单引号）

在A2输入：001（不加引号）

在A3输入：=A1=A2

看到结果了吗？**FALSE**。

它们看起来都是“001”，但Excel认为它们不同——一个是文本，一个是数字。

**区分方法**：

| **特征** | **文本** | **数字** |
| --- | --- | --- |
| 默认对齐方式 | 左对齐 | 右对齐 |
| 能否参与数学运算 | 不能  | 能   |
| 单元格左上角 | 可能有绿色小三角（警告） | 无   |

从系统导出的数据，有时数字会被存成文本格式。如果你的SUM求和结果偏小，很可能就是这个问题。

## 四、运算符——让数据“动起来”

运算符是连接函数和单元格的“桥梁”。

**算术运算符（最常用）**

| **运算符** | **含义** | **示例** | **结果** |
| --- | --- | --- | --- |
| +   | 加法  | \=10+5 | 15  |
| \-  | 减法  | \=10-5 | 5   |
| \*  | 乘法  | \=10\*5 | 50  |
| /   | 除法  | \=10/5 | 2   |

**比较运算符（用于条件判断）**

| **运算符** | **含义** | **示例** |
| --- | --- | --- |
| \=  | 等于  | \=A1=10 → TRUE或FALSE |
| \>  | 大于  | \=A1>10 |
| \>= | 大于等于 | \=A1>=10 |
| <>  | 不等于 | \=A1<>10 |

**文本连接符（&）**

\="广告" & "活动" → "广告活动"

**运算符优先级**

1\. () 括号 ← 永远最先执行

2\. ^ 乘方

3\. \* 和 / ← 乘除

4\. + 和 - ← 加减

如果不确定优先级，就用括号明确指定：

\= (A1+B1) \* C1 → 先加后乘

\= A1 + B1 \* C1 → 先乘后加（结果不同！）

## 五、函数嵌套——公式里面套公式

有时候，一个函数的输出会成为另一个函数的输入。

**需求**：计算ROAS，如果花费为0，不要报错。

**嵌套写法**：

\=IFERROR(销售额 / 花费, 0)

拆解：

1.  内层先算：销售额 / 花费
2.  外层判断：IFERROR(计算结果, 0) → 如果报错就显示0，否则显示结果

**阅读技巧**：从最内层开始，一层层往外读。遇到复杂嵌套，先找到最里面的括号。

## 六、常见错误值——Excel在“告诉你”问题出在哪

公式出错时，Excel会返回以#开头的错误值。它不是想气你，是在帮你定位问题。

| **错误值** | **含义** | **常见原因** |
| --- | --- | --- |
| #N/A | 找不到匹配值 | VLOOKUP/XLOOKUP没找到对应数据 |
| #VALUE! | 值类型错误 | 对文本做了数学运算 |
| #REF! | 引用无效 | 公式引用的单元格被删了 |
| #DIV/0! | 除以零 | 除数为0或空 |
| #NAME? | 函数名错误 | 函数名拼写错了 |

**最常用的处理方式**：

\=IFERROR(原公式, 出错时显示的值)

💡 **在单元格里试试就知道了。**

在A1输入10，B1输入0。在C1写公式=A1/B1，看到什么？#DIV/0!。

现在改成=IFERROR(A1/B1, 0)——看到什么？变成了0。

一样的计算，前者报错，后者优雅地处理了。这就是IFERROR的价值。

## 七、这一章你带走了什么？

回到本章开头：你在C1输入=$A$1+B1，向下拖动时——

- $A$1纹丝不动（**钉死了**）
- B1变成了B2、B3（**跟着跑**）

**你亲眼看到了“引用”的两种行为。**

现在你知道了：

- **相对引用（A1）** → 跟着跑 → 用于每一行独立计算
- **绝对引用（$A$1）** → 钉死了 → 用于所有行共用固定值
- **F4键** → 快速切换 → 写公式时随时按

剩下的运算符、嵌套、错误值，你只需要“知道它们存在”，具体用法遇到时查一下或问AI即可。
