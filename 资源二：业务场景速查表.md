\## 资源二：业务场景速查表



| \*\*老板问的问题\*\* | \*\*用什么做\*\* | \*\*怎么做\*\* |

| --- | --- | --- |

| “各渠道花了多少钱？” | 透视表 | 行→渠道，值→花费 |

| “哪个产品线ROAS最高？” | 透视表 | 行→产品线，值→ROAS |

| “10月Facebook花了多少？” | SUMIFS | 条件：渠道=Facebook，日期=10月 |

| “ROAS>3的有几个？” | COUNTIFS | 条件：ROAS>3 |

| “把600个活动自动分类” | IFS | 按ROAS打高效/正常/待优化 |

| “从广告名中提取地区” | LEFT/RIGHT+FIND | 拆“品牌词-美国”→“美国” |

| “按月汇总趋势” | 透视表+日期分组 | 日期右键→组合→月 |

| “每周自动清洗数据” | Power Query | 记录步骤，以后点刷新 |

| “哪个渠道需要优化？” | KPI | 红黄绿灯，一眼看出 |

| “新数据进来要重做一切” | 刷新  | Ctrl+Alt+F5，全部更新 |
