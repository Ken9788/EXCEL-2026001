\*\*📌 最常用10个函数\*\*



| \*\*函数\*\* | \*\*语法\*\* | \*\*用在哪\*\* |

| --- | --- | --- |

| XLOOKUP | \\=XLOOKUP(找什么, 在哪列找, 返哪列, "缺省") | 匹配产品线、负责人 |

| SUMIFS | \\=SUMIFS(求和列, 条件列1, 条件1, ...) | 统计各渠道花费 |

| COUNTIFS | \\=COUNTIFS(条件列1, 条件1, ...) | 统计活动数量 |

| IFS | \\=IFS(条件1, 值1, 条件2, 值2, ...) | ROAS打标签 |

| IFERROR | \\=IFERROR(公式, 出错显示) | 保护除法计算 |

| LEFT/RIGHT | \\=LEFT(文本, N) / =RIGHT(文本, N) | 拆分广告名称 |

| FIND | \\=FIND("找什么", 文本) | 找到分隔符位置 |

| MONTH | \\=MONTH(日期) | 提取月份 |

| WEEKDAY | \\=WEEKDAY(日期, 2) | 判断星期几 |

| IF  | \\=IF(条件, 成立值, 不成立值) | 二选一判断 |
